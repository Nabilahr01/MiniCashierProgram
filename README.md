# MiniCashierProgram
Fitur ini memungkinkan pengguna untuk mencatat dan menyimpan informasi transaksi pembelian yang terjadi di sistem kasir. Setiap transaksi mencakup:
- ID Transaksi Sebagai identifikasi unik untuk setiap transaksi.
- ID Produk Menghubungkan transaksi dengan produk yang dibeli.
- ID Pelanggan Menghubungkan transaksi dengan pelanggan yang melakukan pembelian.
- Total untuk Menghitung jumlah harga seluruh produk yang dibeli dalam satu transaksi.

Fitur ini berguna untuk mencatat riwayat pembelian, menghitung pendapatan, dan mendukung pencarian transaksi berdasarkan pelanggan atau produk tertentu.
